# Датасет Новости с сайта CNews.ru

Источник: https://www.cnews.ru/

Дата сбора датасета - 26.02.2026 г.

Объём датасета:
- ~1000 HTML страниц
- ~1000 записей с новостями

**Ключевые слова**: текст, СМИ, новости, технологии

### Данные
```
data/
  - database.sqlite -- SQLite3 БД с новостями и HTML страницами
  - robots.txt
  - sitemap.xml -- карта сайта со списком ~1150 новостей (это не все новости сайта)
```

**Таблицы в БД**
- `Url` —  список URL новостей, с отметками о скачивании (скачено или нет). Поля: `id`, `url`, `type`, `state`, `note`. `state` принимает `new`, `downloaded`.
- `pages_content` — Содержимое веб-страниц с новостями; `id`, `content` — HTML код страницы, `url`, `content_hash`, `download_dt`. `content_hash` вычисляется через `blake2s`, `download_dt` хранит ISO-метку времени.
- `News` — таблица с извлечёнными новостями из `pages_content`.

### Лицензия
Исходные тексты и медиа из новостей CNews.ru защищены авторским правом и не находятся под открытой лицензией, доступной для свободного использования.

*Примечание: Поэтому в этом репозитории собранные данные полностью не публикуются*

Цитата с сайта CNews
```
Использование материалов сайта возможно только с письменного разрешения ООО «Синьюс». По вопросам использования материалов издания можно отправить запрос на адрес review@cnews.ru.

Использование материалов сайта в соответствии с п.3 части 1 статьи 1274 Гражданского кодекса РФ без разрешения ООО «Синьюс» запрещается.

Цитирование материалов сайта допускается безвозмездно в объеме, оправданном целью цитирования, но не более 1 тыс. знаков, при этом обязательны:

открытая и доступная для поисковых систем ссылка (без nofollow) на публикацию портала CNews;
текстовое упоминание CNews как источника.
При любом использовании материалов сайта в сети Интернет, как на возмездной, так и на безвозмездной основе, обязательно указание источника в виде гиперссылки на страницу исходного материала на сайте CNews.ru.
```
