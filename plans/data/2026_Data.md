# Задание 2. Скрапер
Создайте скрапер на Python. Скачайте минимум 1000 страниц (новостей, постов, небольших картинок и т.п.) с выбранного вами сайта. Получайте данные из веб страниц. Можно использовать карту сайта.
Выберете такой сайт, у которого нет API для получения данных.

Рекомендуется получать данные с сайта со статическим контентом. Это упростит создание скрапера.

Можно скачать данные из нескольких сайтов, но по 1000+ с каждого.

Скаченные данные могут быть использованы в дальнейшем в работе по кластеризации в курсе Методов МО.


### Требования
- Понятная и логичная структура парсера: структура папок, readme, requirements
- Сохраняйте сперва перечень ссылок, которые нужно обработать, их состояние (обработана, не обработана и т.п)
- Сохраняйте сами веб-страницы (если вы скачиваете веб-контент)
- Предусмотрите возможность продолжить скачивание данных при повторном запуске скрипта
- Отдельно запускайте парсер, который извлечёт из скаченных страниц необходимые данные  (если вы скачиваете веб-контент)
- Учитывайте требования сайта по автоматическому сбору информации, на нарушайте авторские права или законы, в том числе закон о персональных данных
- _Бонус: сохраняйте скаченные данные в СУБД MongoDB_
