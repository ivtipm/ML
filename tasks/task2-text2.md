# Классификация текстов

Решите задачу классификации текстов представленных с помощью нейросетевых представлений (эмбеддинги) текстов и TF-IDF.


1. Изучите и подготовьте тексты. Удалите лишние слова символы и обозначения, знаки пунктуации (только для (TF-IDF)).

- дополнительный балл за 3+ класса

#### TF-IDF (бонус)
2. Подготовьте TF-IDF матрицу: определитесь с размером и составом словаря, критериями попадания слова в словарь, размером n-грамм и т. д.

#### Эмбеддинги
3. Решите задачу классификации текстов используя векторные представления текстов.
4. Сохраняйте полученные эмбеддинги в файл, чтобы с ними было удобно работать. Данные в файле с  эмбеддингами должны быть согласованы с текстовым датасетом.


Для получения эмбеддингов используйте пакет (sentence transformers) transformers,  модель эмбеддингов, запущенную через сервер LLM (например Ollama) или облачную модель. В последнем случае заранее подумайте о том, стоит ли посылать ваши тексты в облако?

Если выбранная модель эмбеддингов поддерживает только короткие тексты, то подавайте в неё текст частями. Потом агрегируйте ответы, например получайте средний эмбеддинг.



 

#### Классификация
В качестве моделей используете логистическую регрессию, метод опорных векторов или *модели на основе байесовского классификатора (дополнительно)* или другие. Подберите гиперпараметры.

4. Оцените модели на тестовой выборке. Сравните результаты.
	- Дополнительный балл за сравнение с учётом гиперпараметров, способов подготовки текста и в виде таблиц с сервиса WandB или MLFlow.
5. Напишите выводы о проделанной работе и полученном  опыте.


#### Классификация (бонус)
Используйте LLM для классификации.\
Напишите промпт, с описанием задачи. Подавайте классифицируемый текст вместе с промтом.

Рекомендуется включить для LLM форматированный вывод, например в JSON чтобы ответы всегда были одного вида. 

Экспериментируйте с различными LLM, температурой и промтом чтобы достичь наилучшего качества.

Пример: [../NLP/text_classification_llm.py](../NLP/text_classification_llm.py)
